# 污水處理設施計算系統 v2.4

Wastewater Treatment Calculator - 環境工程廢水處理設計計算工具

## 功能特色

- ✅ 65+ 事業類別自動帶入申報項目
- ✅ 16 種處理單元預設削減率
- ✅ 質量平衡計算
- ✅ 額外進流管理（RAS、化學藥劑、上清液等）
- ✅ EMS 格式表格
- ✅ 自訂單元名稱、處理線名稱、進出水編號
